# MessageForge

A lightweight and flexible distributed messaging library designed to simplify the implementation of asynchronous messaging systems. It provides an easy-to-use API for building scalable applications with publisher-subscriber patterns.


## MessageForge.RabbitMQ

[RabbitMQ](https://www.rabbitmq.com/) integration for MessageForge, with publishers, subscribers, connection pooling, and dependency injection extensions.

See the [MessageForge.RabbitMQ README](https://github.com/sspaniel/MessageForge/blob/main/src/MessageForge.RabbitMQ/README.md) for setup and usage.
